# PESCE Campus Information AI 🎓

Interactive Campus Info Chatbot for PES College of Engineering, Mandya

## Features
- 📖 Academic Information
- 💼 Placement Statistics
- 🏢 Campus Facilities
- 📋 Administrative Details

## Setup
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Built with
- Python 3.x
- Streamlit
- JSON Data

## Data
Real PESCE college information collected from official sources.
